## 3. Project Motivation

台灣口腔癌發生率位居全球前列，長年為男性癌症發生率前五名之一。與許多癌症不同，口腔癌與可預防的生活習慣高度相關——尤其是嚼檳榔與吸菸，兩者在台灣皆相當普遍。

根據國健署廣為引用的本土研究（高雄醫學大學葛應欽教授團隊），吸菸者罹患口腔癌的風險為一般人的 18 倍，嚼檳榔者約為 28 倍，飲酒者約為 10 倍；若同時吸菸與嚼檳榔，風險將驟升至 89 倍，若菸、檳榔、酒三者皆有（俗稱「香檳酒」），風險甚至可能高達 123 倍。儘管政府推動口腔癌免費篩檢計畫，針對具危險因子的族群提供定期口腔黏膜檢查，然而篩檢參與率仍有相當大的提升空間，許多高風險民眾未能及時接受篩檢。

依據國民健康署政策，符合條件的民眾可享有免費四癌篩檢補助。口腔癌篩檢的資格為：**30 歲以上且有吸菸或嚼檳榔習慣者，每兩年可接受一次免費口腔黏膜檢查**（原住民則降至 18 歲）。花蓮慈濟醫院依此政策，於門診就醫時主動識別符合資格的個案，適時進行癌篩轉介，讓民眾在就診的同時完成篩檢，無需另行掛號。

然而，並非所有符合資格的就診者都會被即時識別或主動轉介。本專題旨在利用機器學習，根據病患的人口特徵、生活習慣風險因子與歷次篩檢紀錄，**預測哪些人在口腔癌篩檢中可能出現異常結果**。將預測模型整合至門診轉介流程，可協助醫療人員在眾多符合資格的就診者中，優先轉介模型判定為高風險的個案，進而提升早期發現率，改善預後。

更重要的是，若僅依國健署規定的資格條件，對所有符合資格者逐一列印癌篩轉介提示單，符合條件的就診者數量將過於龐大，反而會對門診醫師造成極大的看診負擔；長期下來甚至可能導致醫師對高風險個案的衛教與轉介意願降低，形成「為了不漏掉而提醒所有人，最後卻沒有人被妥善轉介」的反效果。透過機器學習篩選並排序出真正最值得優先關注的高風險個案，可有效降低對診間運作的干擾，讓「主動轉介」真正落實，而非淪為形式上的提醒、徒增醫師負擔。

## 4. Proposed Methods

本專題計畫比較以下四個機器學習模型：

| 模型 | 選擇原因 |
|------|---------|
| **Random Forest（隨機森林）** | 集成式學習，準確率通常較高；可輸出 Feature Importance，揭示哪些風險因子對預測結果影響最大 |
| **Logistic Regression（邏輯迴歸）** | 醫療研究常用的基準模型，結果具可解釋性，可直觀了解各因子對風險的貢獻 |
| **SVM（支援向量機）** | 核函數方法，適合二元分類問題，與線性及樹狀模型形成對照比較 |
| **Decision Tree（決策樹）** | 最易視覺化與解釋的模型，樹狀結構可在簡報中直觀呈現決策邏輯 |

四個模型分屬集成學習、線性模型、核函數方法、樹狀模型等不同演算法類型，可進行有意義的跨類型比較。

### 實驗流程

1. **資料前處理**：計算衍生欄位（年齡、篩檢次數、上次結果、距上次篩檢年數），處理缺漏值，並對類別型欄位進行編碼，確保資料格式符合各模型輸入需求。

2. **資料切割**：將 21,034 筆資料以 80/20 比例隨機切割為訓練集（約 16,827 筆）與測試集（約 4,207 筆），並以分層抽樣（Stratified Split）確保兩組的陽性比例一致。

3. **類別不平衡處理**：口腔癌篩檢資料的陽性率偏低，若不處理，模型傾向將所有個案預測為「正常」仍能獲得高 Accuracy，但實務上毫無意義。本專題採用 `class_weight='balanced'` 設定，使模型在訓練時對少數類別（異常）給予較高權重；同時以 **Recall** 與 **AUC** 作為主要評估指標，而非單純依賴 Accuracy。

4. **模型訓練**：四個模型各自在訓練集上進行學習（fitting），測試集於此階段完全隔離。

5. **模型評估**：訓練完成後，以測試集對四個模型進行預測，計算 Accuracy、Precision、Recall、F1-score、AUC，並繪製 Confusion Matrix 與 ROC 曲線。

6. **比較與選模**：依據評估指標的綜合表現，找出最適合預測口腔癌篩檢陽性的演算法，並分析各模型優劣。

## 5. Dataset Source

- **來源**：花蓮慈濟醫院四癌篩檢資料庫（機構自有資料，非公開下載）
- **性質**：機構自行蒐集之篩檢記錄（Self-collected）
- **授權單位**：花蓮慈濟醫院癌症醫學中心（已取得資料使用授權，使用前需完成去識別化處理）
- **資料擷取**：資料由本人自行從院內資料庫系統匯出（該系統為本人所建立）
- **時間範圍**：近 10 年歷年口腔癌篩檢資料
- **資料筆數**：21,034 筆
- **特徵數量**：10 個特徵 + 1 個標籤（Label）

### 特徵欄位說明

| 特徵 | 說明 | 類型 |
|------|------|------|
| 年齡 | 由出生日期與篩檢日期計算得出 | 數值 |
| 性別 | 男 / 女 | 二元 |
| 篩檢年份 | 本次篩檢的年份；用於捕捉不同年份間的政策推廣、族群組成或篩檢行為差異所形成的時間趨勢 | 數值 |
| 吸菸強度 | 0=從未吸菸；1=已戒；2–5=現在吸菸（依年數與每日數量分級） | 有序數值 |
| 嚼檳榔強度 | 0=從未嚼；1=已戒；2–5=現在嚼（依年數與每日數量分級） | 有序數值 |
| 是否原住民 | 是 / 否 | 二元 |
| 自覺口腔不適 | 有自覺症狀 / 無 | 二元 |
| 累計篩檢次數 | 該病患截至本次的歷史篩檢累計次數（程式計算） | 數值 |
| 上次篩檢結果 | 前一次篩檢是否異常（程式計算，初次填 -1） | 二元 |
| 距上次篩檢年數 | 與前一次篩檢相隔幾年（程式計算，初次填 0） | 數值 |

### 標籤（Label）

| 值 | 意義 | 原始代碼 |
|----|------|---------|
| 0 | 正常 | `00`（無明顯異常） |
| 1 | 異常 | 其他代碼（如白斑、紅斑、紅白斑、疑似口腔癌等） |

> 備註：資料使用前將進行去識別化處理，以匿名流水號取代個人識別資料。

## 6. Expected Results

1. **模型效能比較**：以 Accuracy、Precision、Recall、F1-score、AUC 比較四個模型，找出最適合預測口腔癌篩檢異常的演算法。本研究假設 Random Forest 的綜合表現將優於其他模型，理由是醫療篩檢資料可能含有非線性關係與特徵間的交互作用（如吸菸與嚼檳榔並存時風險急遽升高），而集成學習對此類結構具有較強的捕捉能力；此假設將透過實驗結果加以驗證或修正。

2. **特徵重要性分析**：利用 Random Forest 的 Feature Importance，分析各特徵對預測結果的貢獻程度。本研究假設生活習慣相關因子（嚼檳榔強度、吸菸強度）及歷史篩檢紀錄（上次篩檢結果）將具有較高的重要性，但實際排序將以實驗結果為準。

3. **風險因子洞察**：分析吸菸與嚼檳榔的強度分級（0–5）與篩檢異常率之間的梯度關係，預期呈現「強度越高、陽性率越高」的規律。

4. **視覺化輸出**：各模型的 Confusion Matrix、ROC 曲線比較圖、Feature Importance 長條圖，以及異常位置分布圖（如舌、頰黏膜等部位的發現頻率）。

5. **實務貢獻**：期望驗證僅憑人口特徵與生活習慣等基礎資料，是否已足以透過機器學習有效識別口腔癌篩檢高風險個案，並探討將預測模型整合至門診轉介流程的可行性。
